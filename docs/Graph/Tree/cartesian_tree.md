## 概要
Cartesian Tree を求めるアルゴリズム。

## 使い方
`cartesian_tree(arr: Sequence[int]) -> List[int]`  
dinstinct な数列 `arr` から誘導される Cartesian Tree を求め、各頂点の親の配列を返す。計算量 $O(V)$
